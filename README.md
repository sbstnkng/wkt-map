# WKT Map
Visualize [WKT](https://en.wikipedia.org/wiki/Well-known_text_representation_of_geometry) shapes on [OpenStreetMap](https://www.openstreetmap.org).

## Installation
```
npm install
```

## Test application
```
npm test
```

## Build application
```
npm run build
```

## Run application
```
npm start
```
